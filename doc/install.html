Installation instructions for Jack
  • download jack.
  • uncompress it: 
    $ gunzip jack-[VERSION].tar.gz
  • untar it: 
    $ tar xvf jack-[VERSION].tar
  • change directory: 
    $ cd jack-[VERSION]
  • follow instructions from file "INSTALL"
  • run 
        $ jack --help
    or 
        $ jack --longhelp
    or read the supplied manpage.
  • you may want to enable proxy support via the http_proxy environment variable 
        $ http_proxy=http://cache.sld.tld:3128/
   $ export http_proxy
