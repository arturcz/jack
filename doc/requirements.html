System requirements for Jack
Jack has been developed using Linux, but I think it should work with other unices, too, as long as it resembles posix (has ptys, ...). If it doesn't work, tell me!

In order to run Jack, you must have the following installed:

  • python: at least version 2.2. You also need the following modules (they are all part of the python source distribution but some may not be installed by default:
    • termios
    • sndhdr
    • urllib
    • pty
    • wav
    If it doesn't run, check these first!
  • Ben Gertzfield's ID3 module.

Supported DAE utils (ripper):

  • cdparanoia
  • cdda2wav
  • dagrab
  • cdrdao (jack supports ripping from an cdrdao-generated disk-image)
  • tosha

Supported audio encoders:

  • ogg vorbis
  • lame
  • GOGO-no-coda
  • FLAC, the Free Lossless Audio Codec
  • musepack a.k.a. MPC
  • mp3enc
  • xing mp3 encoder
  • l3enc
  • bladeenc

Optional:

  • pyid3lib for ID3v2 support
  • pyogg and pyvorbis for tagging ogg vorbis files
  • xtermset
